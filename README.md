# 長庚大學畢業學分工具 API
### Technologies Used
* Python 3
* Flask
* gevent
* gunicorn
* docker
### Setup
* install docker
* build docker image and run container with command line ```sh build.sh```
* connect api with ```http://localhost:9000```
